深度学习的历史趋势

  • 人工智能的一个关键挑战：
    • 将非形式化的知识传达给计算机
    • 如何将非形式化的知识表示出来，对机器学习算法的性能会产生巨大的影响，如下图所示用极坐标表示数据比用笛卡尔坐标系表示数据更容易用线性模型将数据分开

  • 表示学习(Representation Learning)

    • 对于许多任务，同一类物体的变体(Variation)很多而且在不同场景下物体的表现形式不同，所以我们很难知道要手工提取哪些特征，因此引出表示学习
    • One solution to this problem is to use machine learning to discover not only the mapping from representation to output but also the representation itself. This approach is known as representation learning.
    • 表示学习很难提取高层次、抽象的特征，所以引出了深度学习
    • 深度学习：通过其它比较简单的表示来表达复杂表示，解决了表示学习中的核心问题
    • 典例：自编码器(Autoencoder)

  • 深度学习(Deep Learning)

  • 下图 1_4 和 1_5 展示了不同 AI 学科之间的关系以及每个学科是如何工作的？


  • 第一次浪潮(1943-1986)
    • 兴起的标志事件：单层感知机的实现，没加激活函数，本质是线性函数
    • 衰落的标志事件：单层感知机不能解决异或(XOR)问题
    • 贡献：随机梯度下降
  • 第二次浪潮(1986-2006)
    • 兴起的标志事件：1986年反向传播算法的提出，并可以用它训练具有一到二个隐层的神经网络
    • 中间的标志事件：万能逼近定理(1989)、卷积神经网路、LSTM、LeNet 的提出
    • 衰落的标志事件：梯度消失、计算能力和数据不足等都导致了网络不能继续深入，于是各式各样的浅层模型相继被提出，比如只有一个隐藏层的 SVM 和 Boosting，核方法和图模型在很多重要任务上都实现了很好的效果，这导致了神经网络热潮的第二次衰退
    • 局限性：对复杂问题其泛化能力受到约束，而且大部分需要人工抽取样本的特征
    • 贡献：One of these concepts is that of distributed representation (Hinton et al., 1986). This is the idea that each input to a system should be represented by many features, and each feature should be involved in the representation of many possible inputs. + 反向传播算法的提出
    • 万能逼近定理
      • 一个前馈神经网络如果具有线性输出层和至少一层具有任何一种‘‘挤压’’ 性质的激活函数（例如logistic sigmoid激活函数）的隐藏层，只要给予网络足够数量的隐藏单元，它可以以任意的精度来近似任何从一个有限维空间到另一个有限维空间的Borel 可测函数
      • 总之，具有单层的前馈网络足以表示任何函数，但是网络层可能大得不可实现，并且可能无法正确地学习和泛化。在很多情况下，使用更深的模型能够减少表示期望函数所需的单元的数量，并且可以减少泛化误差。
  • 第三次浪潮(2006至今)
    • 兴起的标志事件：DBN 网络的提出，可以使用无监督的逐层预训练解决梯度消失问题
    • 中间的标志事件：
      • 2010 年 Xavier 初始化方法的提出
      • 2011 年 ReLU 激活函数的提出，使得可以直接以监督式的方式训练深度神经网络，而不必依赖无监督的逐层预训练，祥见慧神论文讨论。
      • 2012年 AlexNet 和 Dropout 的提出
      • 2014年 GooleNet、VGG、R-CNN等的提出
      • 2015年 BN、Faster R-CNN、ResNet等的提出
      • 2016年和2017年 GAN 的流行(待补充……)
    • 挑战：
      • 无监督学习
      • 深度模型在小数据集上的泛化能力
      • 深度强化学习

###CNN 基础理论和结构演化

  • 激活函数ReLU
  • Because ReLU are nearly linear, they preserve many of the properties that make linear models easy to optimize with gradient based methods.
  • They also preserve many of the properties that make linear models generalize well
  • 稀疏激活更加符合生物学特性
  • 卷积神经网络在深度前馈神经网络的基础上，引入三个基本概念：局部感知野(local receptive fields)，权重共享(shared weightes)和混合(pooling)。
  • 局部感知野意味着网络中的每个节点只会感知到图像中的局部信息，全连接网络会感知输入信号中的所有信息。
  • 共享权重意味着图像中不同位置的局部信息都会被同样的权重进行处理，这大大缩减了卷积网络所需要的参数。在卷积神经网络中，局部感知野及共享权重通过卷积层实现。
  • 混合操作通常紧接在卷积层之后使用，一般用于简化卷积层输出的信息。混合操作利用统计量代替单一值的做法增加了模型对输入图像微小形变(rotation)、位移(平移不变性)、缩放等变换的不变性。 最大池化、平均池化等

###深度学习中的模型优化

  • 批量归一化(Batch Normalization, BN)
  • 输入特征的数值归一化对学习算法的训练稳定性及收敛速度一般都会产生正面作用，可以使训练更稳定地收敛。传统的做法仅仅是将输入数据进行归一化，但是在深层神经网络中，随着每次权重更新，底层神经元的输出、同时也是高层神经元的输入的分布会随之改变。批量归一化在深度神经网络不同层的神经元中应用归一化，将各个神经元的激活值通过一个批量中的统计量进行归一化后进入更高一层。令T为某层神经元在一个批量下的激活值组成的矩阵，其中每个样本在该矩阵中占据一行，每个隐结点单元在该矩阵中占据一列。我们通过公式$T’ = \frac{T - \mu}{\sigma} $进行归一化。然而，单纯的进行均值方差归一化会引起模型的表达能力下降。将神经元的激活值归一化至0均值，1标准差后，若顶层的激活函数为S形函数，则顶层的输入将会落入S形函数的线性区域。考虑到将多层线性层叠加在一起的表达能力依然是线性，因此，每个批量归一化节点除归一化能力外，一般还增加两个可学习参数$\alpha$ 和 $\beta$、将$T′$变为$\alpha T′ + \beta$。该变形在不减小模型的原始表达能力的同时，可改变隐结点单元中激活值的分布，帮助模型逃离S型函数的线性区域。这两个参数的值通过梯度下降学习到。通过这两个参数，可以确保增加批量归一化后模型不会弱于未增加批量归一化的模型
  • 测试时，$\mu$ 和 $\sigma$ 被替代为在整个训练集中得到的均值方差。通过该设定，我们不需要在测试时通过批次计算$\mu$ 和 $\sigma$ ，因此可以允许模型仅仅对单个样本进行测试。
